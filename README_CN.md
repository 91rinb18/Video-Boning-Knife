# 已正式更名：视频剔骨刀
极简的拖放式图形用户界面，可将音频、视频和字幕流整合到一个单一的 MP4/MKV 文件中，无需重新编码（由 FFmpeg 流复制技术支持）


## ✨ 功能特点
- 拖放多个文件（视频/音频/字幕）
- 选择特定轨道
- 支持 MP4、MKV 容器
- 无损、超快（流复制）

## 💻 系统要求
- Windows 10+（64位）
- FFmpeg（用户自行从 [ffmpeg.org](https://ffmpeg.org/download.html) 下载，添加到系统 PATH）
  ![Main Interface](./screenshots/4-FF.png)


## 🚀 快速开始
1. 下载 FFmpeg，将 `bin` 加到系统 PATH
![Main Interface](./screenshots/"5-Environment variable.png")
![Main Interface](./screenshots/6-path.png)
2. 启动工具 → 拖入视频/音频/字幕文件
3. 选择轨道 → 点击 **封装/导出**
4. 输出完成！
