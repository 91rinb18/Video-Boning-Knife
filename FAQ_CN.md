## 🔧 Technical / 技术问题

**Q: 问：什么是"复用/流复制"（-c copy）？为何不重新编码？ **

答：  复用 = 将音频/视频/字幕轨道无损无延迟地合并为单一文件。`-c copy` 直接复制原始流（类似文件复制）。**限制**：  
- 仅在格式匹配时生效（例如MP4音频必须为AAC格式）。  
- 无法修复同步问题、剪辑片段或转换格式。  
- 不兼容文件请使用"重新编码"模式（速度较慢，可能存在质量损失）。

软件采用拷贝参数速度极快且不损失音质、画质，同时也限制了它快速分割功能不可能完美到关键帧。要完美分割只能重编码。


**Q: 错误：找不到FFmpeg？**  

A: 解压缩FF之后，点“浏览”导航到你解压缩的FF目录，找到bin\ffmpeg.exe，点击打开即可。

**Q: MKV 输出文件的大小是否大于输入文件的大小？**  

A: ：MKV 格式文件会增加元数据的占用空间。 但是mkv兼容性最佳。


**Q: 字幕不显示？**  

A: 选择 SRT/ASS 作为字幕轨道。MKV 支持外部字幕；MP4 则需要嵌入（重新编码）。蓝光Blu-ray的图形字幕PGS，不能封装到MP4。本工具的拷贝功能并不能让它成为文本的srt。


**Q: ：“软字幕”和“嵌入字幕”有什么区别？为什么你们的软件不提供“嵌入字幕”这一功能呢？**

A: 软字幕是把字幕文件附加到视频文件中。烧录字幕是把字幕渲染到画面里面。（即硬编码字幕）。烧录需要对视频重新编码，这与“快速且无损”的 -c copy 选项相冲突。


**Q: 为何将 SRT 文件添加到 MP4 文件中会失败？**

A: SRT（子字幕表）并非 MP4 格式所支持的官方字幕格式。通常需要将其转换为 mov_text 格式，才能在 MP4 中作为字幕轨道发挥作用。


**Q: 为什么 Windows 系统/杀毒软件会将该可执行文件标记为恶意程序？**

A: 众多用户在 Python 的官方问题反馈中提到，VirusTotal/Defender 会错误地将一个文件/新版本识别为病毒。该工具不会上传文件，而是进行离线操作。我目前正在提交关于这些防病毒误报的报告。
