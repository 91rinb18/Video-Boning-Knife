# 更新日志

**简体中文** | [English](./CHANGELOG.md)

## [1.2.18] - 2026-02-14
**核心关键词：稳定性、本地化、终极打磨**

### 优化
- 深度本地化：统一了“分离/封装/分割”等专业术语。

### 修复
- 脚本引擎增强：修复了在特定复杂文件路径下生成脚本可能出现的转义字符错误 (Syntax/Escaping fix)。

### 体验
- 智能目录管理：防止路径报错。

### UI
- 布局自适应修复：解决了在高 DPI 缩放屏幕下，“关于”窗口底部按钮可能被遮挡的问题。

## [1.2.12] - 维护更新
**核心关键词：多线程、日志、防假死**

### 性能
- 异步分析引擎：引入多线程 (Multi-threading) 处理文件分析，彻底解决导入大文件（如 4K HDR 电影）时界面“白屏/假死”的问题。

### 功能
- 实时日志系统：新增执行日志滚动窗口，实时回显底层 FFmpeg 进程状态，进度可控。

### 修复
- 容器兼容性补丁：修复了 PS 容器识别崩溃的问题。
- 解决部分 Windows 系统下回车换行符导致的指令解析异常。

## [1.2.5] - 细节完善
**核心关键词：图标、路径兼容、分割警告**

### 优化
- 路径兼容性：重构路径解析算法，完美支持包含空格、特殊符号的 Windows 文件路径。

### 交互
- 智能引导：在“无损分割”模块增加关键帧警告（Keyframe Warning），提示用户无损模式下的切割精度限制。

### 修复
- 修复了特定操作下颜色变量引用错误 (NameError) 导致的闪退。

## [1.2.0] - 重大更新 (Midnight Nebula)
**核心关键词：全新 UI、拖拽支持、深度分析**

### 重构
- 视觉焕新：引入 "Midnight Nebula" (午夜星云) 设计语言，采用深蓝磨砂玻璃风格，大幅降低夜间使用视觉疲劳。

### 交互
- 拖拽流 (Drag & Drop)：支持将文件直接拖入主界面或特定轨道槽位。

### 核心
- FFprobe 深度集成：拖入文件即刻自动分析编码格式 (HEVC/AVC/AAC等)，无需手动查看属性。

### 修复
- 解决了界面布局在不同分辨率下的塌陷与抖动问题。

## [1.1.5] - 逻辑增强
**核心关键词：智能容器、编码映射**

### 智能
- 自动容器切换：封装 (Mux) 模式下，若检测到 MP4 不支持的流，自动将输出容器切换为兼容性更好的 MKV。

### 核心
- 编码映射表：建立了完整的 Codec-后缀映射库，大幅提升分离成功率。

### 修复
- 修复了时间戳乱序问题。

## [1.1.0] - 功能扩展
**核心关键词：三位一体、分割、封装**

### 新增
- 模块化架构：正式确立 提取 (Extract)、分割 (Split)、封装 (mux) 三大核心功能模块。
- 无损分割：基于 -ss 和 -to 指令实现毫秒级（关键帧限制下）无损切割。
- 混流封装：支持将独立的视频、音频、字幕文件无损封装为单一文件。

## [1.0.0] - 正式版发布
**核心关键词：桌面化、无损哲学**

### 发布
- 架构转型：从 Web 原型正式迁移至 Python 桌面应用架构，实现本地离线运行。

### 核心
- 基于 "Re-encoding Refused" (拒绝重编码) 产品哲学，所有操作用 -c copy 模式，确保画质、音质 100% 无损且速度极快。

### 功能
- 基础轨道提取功能上线。
